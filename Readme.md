
# LLVM IR Syntax Highlighting for Notepad++

Simple LLVM IR syntax highlighting specification for [Notepad++](https://notepad-plus-plus.org/).

`Notepad++` > `Define your own language...` > `Import` > `llvm_ir.xml`
